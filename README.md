# Professional Portfolio

Zihuan Wang's professional portfolio connects mechanical engineering, artificial intelligence, data analysis, and strategic communication.

## Featured Work

**Project 01 — Costco AI Commerce Timeline (2024–2026)**

A professional case study that uses Amazon as the primary benchmark to translate major e-commerce AI developments into strategic priorities for Costco. The project demonstrates emerging-technology research, competitive analysis, strategic synthesis, and visual executive communication.

**Project 02 — Choosing the Right Machine Learning Approach**

A decision-focused comparison of traditional machine learning for structured house-price prediction and deep learning for autonomous-driving perception. The case study connects model choice to data type, explainability, delivery cost, operational risk, and consequences of error.

The portfolio is published with GitHub Pages from the repository's `main` branch.
